# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, NLP, Machine Learning, and Streamlit.

##  Features

- Resume Upload (PDF)
- Skill Extraction
- ATS Score Calculation
- Job Description Matching
- Missing Skills Detection
- Interactive Web Interface


##  Tech Stack

- Python
- Streamlit
- NLP
- scikit-learn
- PyPDF2
- Machine Learning

##  Run Locally

-> Clone Repository

```bash
git clone https://github.com/yourusername/AI-Resume-Analyzer.git
```

-> Open Project Folder

```bash
cd AI-Resume-Analyzer
```

->Install Dependencies

```bash
pip install -r requirements.txt
```

-> Run Application

```bash
streamlit run app.py
```

-> How It Works

1. Upload Resume PDF
2. Paste Job Description
3. AI extracts skills
4. ATS score is calculated
5. Missing skills are displayed



-> Output

- Extracted Skills
- ATS Match Percentage
- Missing Skills Suggestions
